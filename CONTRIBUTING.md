# Contribuire al Progetto / Contributing to the Project

Grazie per l'interesse a migliorare questo repository didattico!
Thank you for your interest in improving this educational repository!

---

## Come Contribuire / How to Contribute

### 1. Fork & Clone

```bash
git clone https://github.com/<tuo-username>/Linux.git
cd Linux
```

### 2. Crea un Branch / Create a Branch

Usa un nome descrittivo:

```bash
git checkout -b feature/aggiunta-appunti-bash
```

### 3. Aggiungi il Materiale / Add Material

- Aggiungi PDF, appunti o risorse nella cartella appropriata
- Aggiorna il `README.md` con la descrizione del nuovo materiale
- Verifica che il contenuto sia **esclusivamente a scopo didattico**

### 4. Commit & Push

```bash
git add .
git commit -m "docs: aggiunti appunti su <argomento>"
git push origin feature/aggiunta-appunti-bash
```

### 5. Apri una Pull Request / Open a Pull Request

Apri una PR descrivendo:
- Cosa hai aggiunto o modificato
- Perché potrebbe essere utile agli altri studenti

---

## Linee Guida / Guidelines

- Il materiale deve essere **solo a scopo didattico** — no copyright violations
- Preferibilmente in italiano o inglese
- I PDF devono essere leggibili e di buona qualità
- Correzioni grammaticali e miglioramenti sono sempre benvenuti
- Rispetta un tono collaborativo e inclusivo

---

## Codice di Condotta / Code of Conduct

Si prega di mantenere un comportamento rispettoso e collaborativo.
Please maintain respectful and collaborative behavior.

---

*Ogni contributo, piccolo o grande, è prezioso per la comunità di studenti.*
*Every contribution, big or small, is valuable to the student community.*
